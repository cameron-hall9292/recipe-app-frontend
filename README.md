# Recipe App

This is the frontend for a recipe app I built using Vue, Typescript, and Vite. The backend was built using NodeJS. I built this app to caputure family recipes. The app is basic crud, but it also has Google Authentication that allows only certain individuals the ability to edit or delete recipes data, but anyone can view/read. In addition, I added a feature to download PDF file versions of each recipe for anyone who wants to save them in their own personal drive or perhaps print off hard copies. The app utilizes responsive design and it looks best on a moblie device although it appears okay on desktop as well. The below screenshots are from a desktop view:
