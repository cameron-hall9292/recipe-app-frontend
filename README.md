# Recipe App

This is a responsive web application built to capture, manage, and share family recipes. 

### Tech Stack
* **Frontend:** Vue, TypeScript, Vite
* **Backend:** Node.js

### Key Features
* **Secure Authentication:** Integrated Google OAuth to restrict editing and deleting capabilities to authorized family members while keeping recipes public for viewing.
* **Responsive Design:** Optimized for mobile devices to ensure easy reading while cooking in the kitchen, while maintaining a clean layout on desktop.
* **PDF Generation:** Allows users to download clean, printable PDF versions of any recipe for offline use or physical archiving.
